# Release Notes - v1.0.0-alpha.5

## What's New
- **App Settings Screen**: A new settings hub to customize your experience.
  - **Theme Selection**: Switch between Light, Dark, or System Default themes.
  - **Date Format Preferences**: Choose how dates are displayed across the app.
  - **Notification Management**: Toggle expiration and general notifications.
  - **Data Management**: Placeholder for future import/export functionality.

## Improvements & Fixes
- **Coil 3 Upgrade**: Migrated image loading to Coil 3 for improved performance and modern API support.
- **Kotlin 2.0 Migration**: Upgraded codebase to Kotlin 2.0.0 and KSP 2.0.0-1.0.21.
- **Stabilization**: Resolved a critical crash after biometric login caused by dependency incompatibilities.
- **Persistence**: Integrated Jetpack DataStore for robust preference management.
- **Search Optimization**: Resolved experimental coroutine warnings and optimized search reactivity.

## Technical Details
- Kotlin: `2.0.0`
- Coil: `3.0.4`
- Compose BOM: `2025.12.01`
- DataStore: `1.1.7`
