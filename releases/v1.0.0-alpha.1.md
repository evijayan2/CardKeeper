# Release Notes - v1.0.0-alpha.1

## Initial Release

### core Features
- **Project Setup**: Initial setup of the CardKeeper Android application.
- **Financial Accounts**: Support for adding, viewing, and managing financial cards (Credit/Debit) with Material 3 UI.
- **Identity Documents**: Basic scanning and management for identity documents.
- **Passport Management**: Added capability to scan and manage passport data.
- **Rewards Cards**: Introduced dedicated support for loyalty and rewards cards.
- **Scanning**: Integrated scanning for payment cards, driver licenses (barcode), and text analysis for data extraction.
- **CI/CD**: Established basic CI/CD pipeline for automated builds.
